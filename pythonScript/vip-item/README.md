## 安装& 使用 ##
### 安装依赖 ###
`pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple --trusted-host mirrors.aliyun.com`
### 获取代码 ###
`git clone ggit@github.com:InfiniteYinux/Python.git`
### 运行 ###
```
cd pythonScript\vip-item
python vip.py
```
